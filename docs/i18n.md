# Basic Language Support

This site supports a lightweight `lang` query parameter for key conversion pages:
- `en` (default)
- `es`
- `pt`
- `ru`
- `ua`

Example:
- `/apply/?lang=es`

## Canonical Strategy

We keep canonical URLs pointing to the base English route (no `lang` parameter) to avoid duplicate-content traps.

## Scope

Translations are intentionally limited to high-impact conversion surfaces:
- Homepage hero + CTAs
- Apply page labels + key helper text
- Packages page top copy + CTAs
- Platforms page primary CTA
