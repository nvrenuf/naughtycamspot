# Model Discovery — New Lane (Pinned Note)

This note captures the current discovery strategy discussion for NaughtyCamSpot.

## Core problem

The old discovery pattern is weakening:
- blast X / Reddit / generic social posts
- hope strangers see them
- hope they click
- hope they spend

This is crowded, easy to copy, and likely to get weaker as public feeds become more AI-filtered and bot-skimmed.

## Core shift

The new lane is not "better posting."
It is **better discovery architecture**.

Working model:
- human curiosity
- repeated identity
- event pull
- trust transfer
- hub capture

## Strategic concept

The model should not look like "hot girl with a link."
She should look like:
- a distinct persona
- a distinct vibe
- a distinct recurring experience
- a world people can enter

## Discovery lanes

1. **Trust-transfer discovery**
   - creator feature swaps
   - relay handoffs
   - themed weeks
   - private-channel mentions
   - adjacent-brand features
   - photographer / stylist / cosplay / nightlife partner exposure

2. **Event-driven discovery**
   - Birthday Week
   - Friday After Dark
   - Cosplay Sunday
   - Fan Vote Night
   - VIP Countdown Show
   - any recurring named event that is more memorable than a generic stream

3. **Model hub as center**
   - not just a link page
   - the front door to the model’s world
   - best photo / vibe
   - live schedule
   - current event
   - where to find her
   - VIP join option
   - best place to start

4. **SFW identity content**
   Public content should sell:
   - charm
   - energy
   - style
   - mystery
   - humor
   - routine
   - anticipation

   Public content should create intrigue, not fulfillment.

5. **Fan referral loops**
   - invite perks
   - friend-access nights
   - VIP referral rewards
   - unlocks for group goals

6. **Adjacent-world partnerships**
   Discovery can overlap with:
   - cosplay
   - alt fashion
   - nightlife
   - beauty
   - tattoos
   - lingerie
   - glamour
   - photography
   - cars / bikes
   - gaming
   - fitness
   - dance
   - festival culture

## New-lane framing options

- persona-first discovery
- event-led model discovery
- hub-centered fan discovery

## Working thesis

The strongest new lane is:

**SFW personality content + recurring named events + trust-transfer collabs + model hub capture**

## Key discussion questions to revisit

1. What should a model hub page actually include?
2. What kinds of SFW / sexy public content create curiosity without looking generic?
3. Which discovery lanes can work with very low budget?
4. What makes one model feel different from the OF crowd in 5 seconds?
